修改VRSpaceShooter/Assets/Done/Don_Scripts/目录下的Done_PlayerController.cs 文件中的serverUrl为本地的IP地址，例如：

private string serverUrl = "http://192.168.199.135:8080/";

运行KinectServer

Build VRSpaceShooter Android平台文件，

安装到手机

点击KinectServer主界面的Start按钮启动摄像头

玩家高举右手，直到KinectServer检测到并将玩家设置为当前用户（用户列表中，该玩家为高亮）

打开手机端游戏

右手平举到胸前进行法炮，同时可以左右移动右手控制水平位置。
